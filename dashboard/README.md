# VAMP Bot Dashboard

A sleek, dark-themed React dashboard for the VAMP Discord bot with red glow effects.

## Setup

1. Install dependencies:
```bash
cd dashboard
npm install
```

2. Start the development server:
```bash
npm start
```

3. Build for production:
```bash
npm build
```

## Features

- Dark black background with red glow theme
- Real-time bot statistics
- Command management interface
- Activity monitoring
- Responsive design

## Tech Stack

- React 18
- React Router
- Pure CSS (no frameworks)
